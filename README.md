# AI_Comments_POC
AI Feedback Analyzer POC: An interactive dashboard built with Streamlit and Google Gemini. It processes customer feedback (from text or CSV) to automatically identify sentiment trends, extract common themes, highlight critical issues, and generate actionable recommendations.

- AI-Powered Analysis: Uses Google Gemini (Flash/Pro) to process unstructured text.
- Interactive Dashboard: Visualizes sentiment distribution and top themes using Plotly.
- Flexible Input: Supports direct text entry, CSV uploads, and custom file parsing.
- Actionable Insights: Automatically flags critical issues and suggests improvements.
- Exportable Reports: Download analysis results as JSON or formatted text reports.
